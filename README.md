# fire-alarm
Simple fire alarm system using smoke detector and temperature sensor with a visual modification for deaf people. Blue led for smoke detection  and red led for high temperature detection.

## Requirements
- Arduino IDE
- Arduino
- Breadboard
- Buzzer
- Gas Sensor
- Temperature Sensor
- Jumper Wires
- LED(x2)
- Resistors(x3)
